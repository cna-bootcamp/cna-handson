## Git 푸시   

- Git Push 하기  
    IntelliJ에서 GitBash 터미널을 엽니다.   

    원격 Git repository에 푸시합니다.  
    ```
    git add . && git commit -m "sc" && git push -u origin sc
    ```

    v6.0.0으로 태깅하고 푸시합니다.  
    ```
    git tag -a v6.0.0 -m "Spring Cloud 연동" 
    git push origin v6.0.0
    ```

- 확인  
    github.com에서 잘 푸시되었는지 소스와 태그를 확인 합니다.
